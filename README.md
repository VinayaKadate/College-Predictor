## ğŸš€ **Quick Start**

### **Prerequisites**
- Node.js 18+ & npm
- Python 3.9+
- Firebase Project
- Google Gemini API Key

### **Installation**
```bash
# Clone repository
git clone https://github.com/VinayaKadate/College-Predictor.git

# Frontend setup
cd frontend
npm install
npm run dev

# Backend setup
cd backend
pip install -r requirements.txt
python app.py
```

### **Environment Setup**
```env
# Frontend (.env)
VITE_API_URL=https://your-backend.onrender.com
VITE_FIREBASE_API_KEY=your_firebase_key

# Backend (.env)
GEMINI_API_KEY=your_gemini_ai_key
JWT_SECRET_KEY=your_secret_key
```
